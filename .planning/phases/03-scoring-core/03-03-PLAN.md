**1. load_calibration(path: Path) -> dict[str, float | int | str]:**


**2. write_calibration(path: Path, alpha: float, beta: float, **kwargs: float | int | str) -> None:**
```
- payload = {"alpha": alpha, "beta": beta, "calibrated_at": datetime.now(timezone.utc).isoformat(), **kwargs}
- path.parent.mkdir(parents=True, exist_ok=True)
- path.open("w") with json.dump(payload, fh, indent=2)
```

**3. apply_hybrid_score(pose: ScoredPose, *, alpha: float, beta: float, n_residues: int) -> None:**
```
- assert pose.vina_score is not None and pose.ad4_score is not None
- pose.entropy_correction = alpha * n_residues
- pose.hybrid_score = pose.vina_score + beta * (pose.ad4_score - pose.vina_score) + pose.entropy_correction
```
Note: this function does NOT validate α/β ranges — validation is in load_calibration.
Mutates pose in place; returns None.

**4. fit_calibration(vina_scores: list[float], ad4_scores: list[float], n_residues_list: list[int], experimental_pkd: list[float]) -> dict[str, float]:**
```
- RT = 0.592  # kcal/mol at 298K (D-09; hardcoded, not a parameter)
- delta_g = [-RT * pkd for pkd in experimental_pkd]
- def objective(params):
    alpha, beta = params
    residuals = [
        (vina + beta*(ad4-vina) + alpha*n) - dg
        for vina, ad4, n, dg in zip(vina_scores, ad4_scores, n_residues_list, delta_g)
    ]
    return sum(r**2 for r in residuals)
- x0 = np.array([0.65, 0.22])
- bounds = [(0.2, 1.2), (0.0, 0.5)]
- result = minimize(objective, x0, method='L-BFGS-B', bounds=bounds)
- alpha, beta = result.x
- hybrids = [v + beta*(a-v) + alpha*n for v, a, n in zip(vina_scores, ad4_scores, n_residues_list)]
- r, _ = pearsonr(hybrids, delta_g)
- rmse = float(np.sqrt(np.mean([(h-d)**2 for h, d in zip(hybrids, delta_g)])))
- return {"alpha": float(alpha), "beta": float(beta), "pearson_r": float(r), "rmse_kcal_mol": rmse}
```

Google-style docstrings on all four functions. Args, Returns, Raises. No bare except.
