```python
from __future__ import annotations
```
First line. Then: argparse, csv, json, logging, sys. Then pathlib.Path. Then the
entropy imports (added via sys.path manipulation or by ensuring the package is installed).

The script reads a calibration CSV that includes pre-computed scores. Schema:
`pdb_id, peptide_sequence, experimental_pkd, vina_score, ad4_score`

Why this schema: fit_calibration() needs vina_scores, ad4_scores, n_residues_list, and
experimental_pkd. The script derives n_residues = len(peptide_sequence) from the CSV.
Running live Vina scoring inside calibrate_alpha.py is out of scope for Phase 3 (Phase 5
CLI will wire calibrate to the full pipeline). For now, calibration requires pre-scored data.

Implement `parse_args(argv=None)` function returning an argparse.Namespace:
- `--training-csv` (required): Path to input CSV (default: data/training_complexes.csv)
- `--output` (required): Path to write calibration.json (default: data/calibration.json)
- `--verbose` / `-v`: Enable DEBUG logging

Implement `main(args=None)`:
1. Parse args if not provided: `args = parse_args(args)`
2. Configure logging (INFO by default, DEBUG if --verbose)
3. Read training CSV with csv.DictReader; collect lists:
   - vina_scores: [float(row["vina_score"]) for row in rows]
   - ad4_scores: [float(row["ad4_score"]) for row in rows]
   - n_residues_list: [len(row["peptide_sequence"]) for row in rows]
   - pkd_list: [float(row["experimental_pkd"]) for row in rows]
4. Log INFO: `"Calibrating on %d complexes from %s", n_complexes, training_csv`
5. Import from hybridock_pep.scoring.entropy:
   `from hybridock_pep.scoring.entropy import fit_calibration, write_calibration`
6. Call `result = fit_calibration(vina_scores, ad4_scores, n_residues_list, pkd_list)`
7. Call `write_calibration(Path(args.output), training_csv=str(args.training_csv), n_complexes=len(pkd_list), **result)`
8. Log INFO: `"Calibrated: α=%.3f β=%.3f r=%.3f RMSE=%.3f → %s", result["alpha"], result["beta"], result["pearson_r"], result["rmse_kcal_mol"], args.output`
9. Validate with load_calibration: `from hybridock_pep.scoring.entropy import load_calibration; load_calibration(Path(args.output))`
   This is a post-write self-check — if α/β converged outside bounds the script aborts here
   with the informative ValueError rather than silently writing a corrupt calibration.

End with:
```python
if __name__ == "__main__":
    main()
```

| fit_calibration output → write_calibration → load_calibration | Post-write self-check: load_calibration validates the just-written file; catches any optimizer boundary convergence issue |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-13 | Tampering | training_complexes.csv injection (corrupt float in vina_score) | mitigate | float() conversion in CSV parsing raises ValueError with row context; script aborts before calling fit_calibration |
| T-03-14 | Tampering | calibration.json post-write tampered before load_calibration check | accept | Self-check is belt-and-suspenders; primary protection is the validation in load_calibration at dock-time |
